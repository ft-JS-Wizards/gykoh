## ✏️ 문제

### [나머지 구하기](https://school.programmers.co.kr/learn/courses/30/lessons/120810)

<br>

## ✅ 나의 풀이

```javascript
function solution(num1, num2) {
  return num1 % num2;
}
```

<br>

## ✅ 다른 사람의 풀이

```javascript
const solution = (num1, num2) => num1 % num2;
```

<br>

## 💁🏻‍♀️ 소감

간단한 연산에서는 화살표 함수를 사용하도록 해야겠다는 생각이 들었습니다.
