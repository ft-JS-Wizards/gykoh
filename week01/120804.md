## ✏️ 문제

### [두 수의 곱](https://school.programmers.co.kr/learn/courses/30/lessons/120804)

<br>

## ✅ 나의 풀이

```javascript
function solution(num1, num2) {
  return num1 * num2;
}
```

<br>

## ✅ 다른 사람의 풀이

```javascript
function solution(num1, num2) {
  return num1 * num2;
}
```

<br>
