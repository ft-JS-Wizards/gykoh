## ✏️ 문제

### [두 수의 차](https://school.programmers.co.kr/learn/courses/30/lessons/120803)

<br>

## ✅ 나의 풀이

```javascript
function solution(num1, num2) {
    return num1 - num2;
}
```

<br>

## ✅ 다른 사람의 풀이

```javascript
function solution(num1, num2) {
    return num1 - num2;
}
```

<br>

## 💁🏻‍♀️ 소감

- 연산자에 유의해서 두 수의 차를 구해주었습니다.
